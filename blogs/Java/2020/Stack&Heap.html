# Java中的栈和堆的区别

人们常说堆栈堆栈，堆和栈的区别在哪？什么是堆？什么是栈？它们里面存放什么？

# 栈（Stack）

​ 先说说Java中的栈，栈内存中一般存放的是基本数据类型（八大）、局部变量、引用类型。每个线程都会有一个独立的栈空间，所以线程之间是不共享数据的。栈的操作遵循着先进后出、后进先出的原则。它的优点存取速度比堆要快，仅次于直接位于CPU中的寄存器。栈中的数据可以共享。它的缺点是存在栈中的数据大小与生存期必须是确定的，缺乏灵活性。

# 特列：

①方法的引用参数，在栈空间分配一个地址空间，并指向堆空间的对象区，当方法调用完成后从栈空间回收。

②局部变量new出来之后，在栈控件和堆空间中分配空间，当局部变量生命周期结束后，它的栈空间立刻被回收，它的堆空间等待GC回收。

# 堆（Heap）

​ 堆中存放的是new出来的对象、数组。可以动态的分配内存大小、比较灵活，生存期也不必事先告诉编译器，Java的垃圾收集器会自动收走这些不再使用的数据。在堆中分配的内存，由Java虚拟机的自动垃圾回收器来管理。缺点由于要在运行时动态分配内存，存取速度较慢。

# 特列：

​ ①引用数据类型（需要用new来创建），既在栈控件分配一个地址空间，又在堆空间分配对象的类变量。

public static void main(String[] args) {
    String str = "123";
    String str1 = new String("abc");
}

String str1 = new String("abc");这里面： "abc"表示栈中的一个存储空间中的一个数据，new String("abc")表示存在于堆中的一个对象，这个对象的值为"abc"，String str1则表示栈中定义的一个取了new String("abc")在堆中的首地址的一个特殊变量，也就是：str1 成了引用变量，相当于一个别名。然而String str = "123"这个时候str是存放在栈中的。

# 实例：

package com.wh.test;

public class Test {
    static class Person {
        //年龄
        int age;

    }

    public static void main(String args[]) {

        int age = 20;
        //人类初始化
        //此时person是存放在栈中然后new Person()是存放在堆中
        Person person = new Person();
        person.age = 20;

        cs(age, person);
        System.out.println("age=" + age + ",  person.age = " + person.age);

    }

    static void cs(int age, Person person) {

        age = 11;
        person.age = 21;
        System.out.println("age=" + age + ",  person = " + person);

    }

}
